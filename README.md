# Blur Lasers — Roblox Laser System Panel

A comprehensive laser control panel UI built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 4 + shadcn/ui
- **Animations:** Framer Motion
- **State:** Zustand
- **Database:** Prisma (SQLite)

## Setup

```bash
npm install
npm run dev
```

## Deploy

This project is optimized for Vercel deployment. Simply connect your GitHub repo to Vercel.

## Features

- **Landing Page** with animated entry
- **Control Panel** — Master on/off, fade, hold, effects grid, tilt/pan direction
- **Player Management** — Whitelist, kick, role-based permissions
- **Group System** — Create and manage laser groups (fixture or individual)
- **Customisation** — Color wheel, faders, odd/even, left/right, patterns
- **Hub Panel** — Browse community effects, search, filter, add to custom
- **Effect Panel** — Frame-by-frame editor with canvas, timeline, presets
- **Position Section** — One-time toggle buttons for stored positions
- **Info Panel** — Stats and settings
- **Easter Eggs** throughout the UI
